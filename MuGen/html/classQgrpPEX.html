Detailed Description

Student- \(t\) weights for PEX.

Implements the weight updating for van Dyk and Meng's [7] PEX scheme. The multiplicative redundant parameter \(\alpha\) is a member of this class and is updated internally. \(\alpha\) is initialized deterministically to 1.0.
