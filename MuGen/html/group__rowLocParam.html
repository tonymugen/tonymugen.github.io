Detailed Description

A hierarchy of classes that refer to rows of location parameter matrices. These classes are internal to Grp classes and are not directly declared by the user of the library. They are typically updated with samples from the multivariate normal distribution. Because the methods are used for much of the computation, they are implemented for speed and almost no chacking for errors is done. The assumption is that the encapsualting classes will do all of that correctly. Constructors are used to set initial values.
